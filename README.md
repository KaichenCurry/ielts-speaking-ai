# ielts-speaking-ai
# 雅思口语 AI 助教系统

> 面向雅思口语教师的 AI 辅助教学工具 —— 课后作业布置、自动评测、数据存档、周报分析，让老师专注于真正的教学干预。

🇨🇳 [中文介绍](#中文介绍) | 🇬🇧 [English](#english)

---

## 🎯 产品定位

**目标用户：雅思口语教师**

这不是一个学生直接使用的 App，而是一个**给老师用的 AI 助教系统**。老师通过 Telegram Bot 布置作业，学生在家完成练习，系统自动完成评分、反馈、存档全部工作。

---

## 💡 核心价值

| 老师痛点 | 我们的解决方案 |
|---------|---------------|
| 重复性评分工作繁重 | AI 自动评分，老师从"评分员"变成"教学干预者" |
| 作业反馈慢，学生错失记忆窗口 | 学生答题后即时收到逐句反馈 |
| 学生数据散落，难以追踪 | 全部自动写入 Notion，永久留存成长轨迹 |
| 难以批量管理学生进度 | 周五自动推送班级周报，全景一览 |
| 题库更新耗精力 | 每周三、六自动更新题库，覆盖66套真题 |

## 🚀 五大核心功能

### 1️⃣ 一键布置作业
```
老师发送 /题目 Test XX
    ↓
Bot 自动解析并发送 Part 1/2/3 全部题目
    ↓
学生在家随时练习
```
老师只需一条指令，66套真题题库随时调用。

### 2️⃣ AI 自动评测
```
学生语音 → Whisper 转写 → RAG 增强 → MiniMax 评分
    ↓
逐句多维度反馈（语法/词汇/时态/逻辑/思路）
    ↓
Band Score 计算（Part1×30% + (Part2×40%+Part3×60%)×70%）
```
评分过程全自动，老师无需参与。

### 3️⃣ 即时逐句反馈
```
答题完成后 → 学生立即收到逐句反馈
    ↓
每句话对应：
• 语法问题（主谓一致、从句使用...）
• 词汇问题（Chinglish、高分替换...）
• 时态问题（过去时、现在时...）
• 逻辑问题（因果、转折...）
• 思路问题（举例、深度...）
```
不是给个冷冰冰的分数，而是手把手告诉学生哪里有问题、怎么改。

### 4️⃣ Notion 数据存档
```
每次练习 → 自动写入 Notion 作业库
    ↓
老师随时可查：
• 单个学生的历史答题记录
• 班级整体练习频次
• 常见错误类型分布
```
学生数据全部结构化存档，支持后续分析和教学决策。

### 5️⃣ 班级周报推送
```
每周五 18:00 → 自动推送周报到群
    ↓
内容包括：
• 本周班级 Band 分布
• 常见错误类型 TOP5
• 下周教学建议
```
老师无需手动统计，周五早上打开手机就能看全景。

## 🧠 AI 能力设计

### 多模型协同

| 环节 | 技术选型 | 作用 |
|------|---------|------|
| 语音识别 | Whisper | 学生语音 → 文字 |
| 评分推理 | MiniMax | 5维度评分 |
| 上下文增强 | RAG | 融合历史错题 |
| 数据存储 | Notion | 题库/作业/错题本 |

### 评分维度

| 维度 | 关注点 |
|------|--------|
| 语法 | 主谓一致、从句、介词 |
| 词汇 | Chinglish、高分词汇 |
| 时态 | 过去/现在/完成时 |
| 逻辑 | 因果、转折、跑题 |
| 思路 | 举例、深度、观点 |

## 👨‍🏫 老师操作指南

### 布置作业
```
/题目 Test 07
```
一条指令，Part 1/2/3 全部题目自动发给学生。

### 查看学生作业
打开 Notion 作业库 → 按学生/按时间筛选 → 查看详细逐句反馈

### 纠正学生错误
```
/纠正 [正确表达]
```
系统自动收录到错题本，下次评分时 RAG 会自动关联。

### 查看周报
每周五 18:00 自动推送到群，老师无需任何操作。

## 🚀 快速部署

### 环境要求
- Python 3.8+
- Telegram Bot Token
- MiniMax API Key
- Notion Integration Token
- OpenAI Whisper

### 配置
```bash
git clone https://github.com/KaichenCurry/ielts-speaking-ai.git
cd ielts-speaking-ai

pip install openai notion-client python-telegram-bot whisper

export TELEGRAM_BOT_TOKEN="your_token"
export MINIMAX_API_KEY="your_key"
export NOTION_TOKEN="your_notion_token"
```

### 运行
```bash
python scripts/ielts_flow.py
```

## 🇨🇳 中文介绍

### 雅思口语 AI 助教系统

面向雅思口语教师的 AI 辅助教学工具，核心功能：

- **一键布置作业**：66套真题，随时调用
