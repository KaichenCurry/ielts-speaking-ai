# 雅思口语个性化逐句反馈 Prompt

> **核心理念**：没有标准答案。只有学生自己的答案。
> 基于学生说的每一句话，给出个性化的语法/词汇/时态/逻辑/思路反馈。

---

## 反馈维度（5个核心维度）

| 维度 | 说明 | 关注点 |
|------|------|--------|
| **语法** | 句子结构是否正确 | 主谓一致、从句使用、介词搭配 |
| **词汇** | 用词是否地道、准确 | Chinglish、同义词替换、高分词汇 |
| **时态** | 时态是否准确 | 过去时、现在时、完成时、混合时态 |
| **逻辑** | 逻辑是否连贯 | 因果关系、转折、层次感、跑题 |
| **思路** | 论证是否有力 | 举例是否具体、深度够不够、是否有独特观点 |

---

## 核心原则

**逐句分析，不漏一句。**

对学生答案中的**每一句话**进行五个维度的个性化反馈：
- ✅ 做得好的地方（具体指出哪好）
- ❌ 需要改进的地方（针对学生实际说的内容）
- 💡 具体的改进建议（可以给出优化后的句子）

**不要对照任何标准答案**，只针对学生实际说出口的内容进行反馈。

---

## 逐句反馈标准格式（所有Part统一）

每句话按以下格式输出，不漏一句：

```
原句： "{学生原话}"

• 语法：❌/✅/⚠️ + 具体解释（如有错：原词 → 正确词，或具体语法问题）
• 词汇：❌/✅/⚠️ + 具体解释（如有错：原词 → 正确词）
• 时态：❌/✅/⚠️ + 具体解释
• 逻辑：❌/✅/⚠️ + 具体解释

───

原句： "{下一句学生原话}"

• 语法：❌/✅/⚠️ + 具体解释
• 词汇：❌/✅/⚠️ + 具体解释
• 时态：❌/✅/⚠️ + 具体解释
• 逻辑：❌/✅/⚠️ + 具体解释

───
```

### 示例

```
原句： "I don't think there is a simple magic number, but most experts, the keys can start learning the art as soon as they sit up and have a crane,"

• 语法：❌ "most experts, the keys" → "most experts say kids"
• 词汇：❌ "a crane" → "a crayon"（蜡笔）
• 逻辑：✅ 开头观点合理

───

原句： "usually around like the top to 18 months."

• 词汇：❌ "top to" → "about" 或 "around"
• 逻辑：⚠️ 想表达"大约12到18个月"但措辞不清

───

原句： "So I think that other around age 5 or 6, it start up to move past simple crabs"

• 词汇：❌ "other" → "others" 或删掉
• 语法：❌ "it start up" → "they start to"
• 词汇：❌ "crabs" → "scribbles"（涂鸦）
• 逻辑：✅ 年龄分段观点好

───

原句： "and begin to draw things that actually resemble the real world."

• 语法：✅ 这句正确！
• 词汇：✅ "resemble the real world" 表达准确

───

原句： "The keys to keep it fun and low pressure so they can lose their nature creative spark."

• 语法：❌ "The keys to keep" → "The key is to keep"
• 词汇：❌ "lose their nature" → "lose their natural"
• 逻辑：⚠️ 逻辑反了 — 应该是"保持有趣低压"来"保护"创意火花，不是"失去"

───

💡 总体评价：Band 5.5 — 有具体的年龄建议（18个月、5-6岁），但关键词汇错误（crane/crayon, crabs/scribbles）和语法问题影响表达。
```

### 符号说明

| 符号 | 含义 |
|------|------|
| ✅ | 这点做得很好 |
| ❌ | 明显错误，需要纠正 |
| ⚠️ | 有小问题或可优化空间 |

## Band Score 计算

```
═══════════════════════════════════════
🏆 Band Score
═══════════════════════════════════════
Part 1 均分：X.X
Part 2：X.X  （权重 40%）
Part 3 均分：X.X  （权重 60%）

综合 Band Score：Part1×30% + (Part2×40%+Part3×60%)×70% = X.X / 9.0
```

> Part 1 均分单独计算后，按 30% 权重计入综合分。

## 周报 AI 分析 Prompt

学生提交周报数据时，用以下 prompt 生成深度分析：

```
你是雅思口语教学助手，生成本周深度学情报告。

## 学生信息
学生：{学生昵称}
本周练习：{次数}次
平均Band：{均分}
Band范围：{最低}~{最高}

## 各次练习Band
{列表：快捷ID | P1 P2 P3 综合}

## 本周错误记录
语法错误（{n}次）：{逐句列出}
词汇错误（{n}次）：{逐句列出}
时态错误（{n}次）：{逐句列出}
逻辑问题（{n}次）：{逐句列出}
思路问题（{n}次）：{逐句列出}

## 本周错题纠正
{列表}

## 产出要求
生成深度周报，包含：
1. 本周表现综述（1-2句）
2. 各维度详细分析（语法/词汇/时态/逻辑/思路，每个2-3句）
3. 最需突破的1-2个具体问题（给出例子和改进建议）
4. 下周练习重点（1-2条具体可行建议）

注意：只基于真实数据，不编造。表现好的维度也要明确指出。
```
